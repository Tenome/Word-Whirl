# Word Whirl 
A fork of VTT's existing Word Whirl (So Clover), with an [expanded English dictionary](https://boardgamegeek.com/thread/3638738/new-large-word-set). Naturally, the difficulty of the game has also greatly increased. Time to learn some new words!
